# 🐱 FEEDING - 貓舍繁殖紀錄管理系統

這是一個專為貓舍設計的繁殖紀錄管理系統，提供單機/區網的網頁端操作，並支援動態印章、批次匯出 EXCEL 以及資料自動同步功能。

## 🚀 系統功能特色
- **母貓/公貓清單管理**：自動篩選、切換列表，並支援全選與多選操作。
- **繁殖紀錄統計**：追蹤配種日期、預產期、胎次與子代總數。
- **自訂報表與印章**：支援公司大章與負責人小章上傳（無容量限制），並可自訂印章位置（左上、右上、左下、右下、或自訂 EXCEL 儲存格如 `A1`）。
- **批次 EXCEL 匯出**：勾選多隻貓咪後，一鍵匯出單一 EXCEL 檔案（內含多個獨立分頁）。
- **列印最佳化**：自動適應一頁寬度設定。

## 🛠️ 技術架構
- **Frontend**: HTML5, Vanilla JavaScript, CSS, `exceljs` (處理 EXCEL 產出)。
- **Backend**: Node.js + Express (處理檔案存取與高容量圖片 Base64)。
- **Process Manager**: PM2 (背景常駐運行)。

## 💻 如何啟動專案

### 1️⃣ 前置需求
請確保電腦中已安裝 [Node.js](https://nodejs.org/)。

### 2️⃣ 安裝依賴套件
開啟終端機，進入專案根目錄後執行：
```bash
npm install
```

### 3️⃣ 啟動伺服器
**方法一：一般啟動 (開發/測試用)**
```bash
node server.js
```

**方法二：背景常駐啟動 (正式環境推薦)**
如果你有安裝 PM2，可以使用以下指令讓伺服器在背景持續運行：
```bash
npx pm2 start ecosystem.config.js
```
*(啟動後可使用 `npx pm2 list` 查看運行狀態)*

### 4️⃣ 開啟網頁
伺服器啟動後，請打開瀏覽器並前往：
👉 **[http://localhost:3000](http://localhost:3000)**

*(若需讓外網存取，可搭配 ngrok 使用)*
